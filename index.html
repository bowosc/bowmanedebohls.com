About this site

How's this site set up?

This site's running on an AWS EC2 instance. Right now it's just TS/HTML/CSS, but once I get BowAI up and running there'll be some Python hiding somewhere in here.

Why does this site look like this?

I'm trying to move away from the the sort of portfolio page design that's quickly becoming the default. You know the one, with the hero banner, Bootstrap defaults, and "scroll-to-section" buttons. I think the web's a little better when each site looks at least a little bit different.

This site was built with style in mind, while maintaining ease of navigation and non-basic-ness.
